# 智能招聘 Agent 项目 Code Wiki 生成 Spec

## Why

当前项目已有完整的产品设计规划文档和UI原型,但缺少面向开发团队的结构化Code Wiki文档。开发团队需要一个系统化的技术文档体系,清晰了解项目架构、模块职责、技术选型和实现路径,从而高效推进MVP开发。

## What Changes

- 生成完整的项目架构文档,包括整体架构设计、核心模块职责划分、技术选型说明
- 生成模块依赖关系图和数据流向说明
- 生成项目运行和部署方式文档
- 提供分阶段的开发实施建议

## Impact

- **新增文档:** `docs/code-wiki.md` - 项目整体架构和技术文档
- **新增文档:** `docs/architecture-modules.md` - 模块职责与依赖关系详细说明
- **新增文档:** `docs/deployment-guide.md` - 项目运行和部署指南
- **新增文档:** `docs/development-roadmap.md` - 分阶段开发实施建议

## ADDED Requirements

### Requirement: 项目整体架构文档

系统 SHALL 提供完整的项目架构说明文档,内容包括:

#### Scenario: 架构概览
- **WHEN** 开发团队需要理解项目整体设计
- **THEN** 文档应包含:
  - 系统总体架构图(前端、后端、数据库、Agent核心、工具层)
  - 核心技术栈选型说明(React、FastAPI、LangGraph、PostgreSQL、Redis等)
  - Agent核心推理框架(R-P-R-A-R)的详细说明
  - 四层记忆架构设计(L1-L4)的实现原理

### Requirement: 模块职责与依赖关系文档

系统 SHALL 提供清晰的模块职责划分和依赖关系说明:

#### Scenario: 模块职责说明
- **WHEN** 开发人员需要理解各模块的功能边界
- **THEN** 文档应包含:
  - 8个MVP核心模块的职责说明(任务中心、JD管理、简历工作台、评分报告等)
  - Agent核心5个子系统说明(任务编排器、上下文管理器、工具路由器、结果校验器、异常处理器)
  - 工具层6个核心工具说明(简历解析器、评分引擎、日历服务、JD生成器等)
  - 模块间的调用关系和数据流向图

### Requirement: 项目运行与部署指南

系统 SHALL 提供完整的项目运行和部署说明:

#### Scenario: 开发环境搭建
- **WHEN** 新开发人员需要搭建开发环境
- **THEN** 文档应包含:
  - 前端开发环境配置(React + TypeScript + Ant Design)
  - 后端开发环境配置(Python + FastAPI + LangGraph)
  - 数据库初始化脚本(PostgreSQL + Redis)
  - Agent服务启动流程

#### Scenario: 部署方案说明
- **WHEN** 需要部署到生产环境
- **THEN** 文档应包含:
  - 容器化部署方案(Docker Compose配置)
  - 数据库配置与备份策略
  - 监控告警配置(Prometheus + Grafana)
  - 日志收集与分析方案

### Requirement: 分阶段开发实施建议

系统 SHALL 提详细的渐进式开发实施路线图,采用"模块优先、逐步验证"策略:

#### Scenario: 渐进式开发路线
- **WHEN** 项目启动需要制定开发计划
- **THEN** 文档应包含:
  - 7个开发Stage的详细分解(从JD管理模块开始,逐步扩展到其他模块)
  - 每个Stage遵循"数据层→后端层→Agent层→前端层"的完整闭环
  - 每个Stage包含至少5条量化验收标准
  - 数据层优先的设计理念(先建表→验证数据流→再实现业务逻辑)
  - 模块闭环验证机制(每个Stage独立验收后再扩展)
  - Stage间的依赖关系说明(JD管理→任务管理→简历解析→评分→Agent→推送→看板)
  - 团队分工建议和协作模式
  - 关键技术决策点和风险预案

## 文档生成原则

1. **结构化优先:** 所有文档采用清晰的层级结构,便于快速定位
2. **图文并茂:** 核心架构使用Mermaid流程图和架构图辅助说明
3. **实用性导向:** 重点说明"如何实现"而非"为什么设计"
4. **渐进式披露:** 从整体架构到模块细节,逐层深入
5. **可执行性:** 提供具体的代码示例和配置模板
6. **版本追踪:** 每个文档标注版本号和更新日期
